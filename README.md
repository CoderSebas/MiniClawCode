# MiniClawCode

[![CI Pipeline](https://github.com/your-org/mini-claw-code/actions/workflows/ci.yml/badge.svg)](https://github.com/your-org/mini-claw-code/actions/workflows/ci.yml)

A lightweight AI coding assistant built from scratch, inspired by learn-claude-code.

## Features

- AI-powered code generation and refactoring
- Context-aware suggestions
- Fast and lightweight
- Extensible plugin system
- Docker support

## Installation

```bash
pip install mini-claw-code
```

## Quick Start

```bash
miniclaw --help
```

## Development

```bash
git clone https://github.com/your-org/mini-claw-code.git
cd mini-claw-code
pip install -r requirements.txt
python -m pytest
```
